# 微信聊天记录可视化

一个基于HTML和CSS的微信聊天记录可视化项目，使用苹果风格的设计，展示聊天记录的统计和分析数据。

## 功能特点

- 苹果风格的UI设计
- 响应式布局
- 聊天记录统计分析
- 热门话题展示
- 活跃用户排行
- 词云展示

## 技术栈

- HTML5
- CSS3
- JavaScript

## 使用方法

1. 克隆项目到本地
2. 在浏览器中打开`code.html`文件
3. 查看聊天记录的可视化展示
