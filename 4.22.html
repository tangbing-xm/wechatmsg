排查步骤：

  • 检查网络/代理设置（全局模式）。
  • 尝试部署到 Vercel。
  • 核对 Endpoint/Model Name。
  • 确认 API 调用限制。
  • 尝试其他 API 平台（如 OpenRouter）或免费资源。
